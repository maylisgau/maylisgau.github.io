Collection kokophile - affiches

Ce projet a pour but de présenter la collection d'un kokophile en affiche.

Description

Koko est un mot finnois qui signifie taille. L'affiche de présentation de cette collection montre plusieurs expérimentations graphiques créées à partir de ces objets. Les autres affiches sont des compositions créées avec le vocabulaire obtenu.
