Revue The One

Ce projet a pour but d'avoir une revue quotidienne qui informe des dernières nouvelles des membres du groupe One Direction.

Description

La palette de couleurs de cette revue est basée sur les couleurs associées aux membres du groupe. La modification entre les numéros se fait par les motifs plutôt que par les couleurs. Les informations étant disponibles sur les réseaux sociaux, cette revue est imaginée comme un objet pour marquer une date spéciale pour l'acheteur.
