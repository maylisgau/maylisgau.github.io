Couverture de Louis Braille de Margaret Davidson

Ce projet est une propositioin de couverture pour attirer les lecteurs.

Description

L'attention est portée sur le braille, un moyen d'écriture inventé par Louis Braille pour les mal-voyants. La typographie Tiny5x3 de Jack Halten Fahnestock est mise en parallèle avec le braille. Les inserts sont des jeux pour apprendre le braille.
